# Hostel-Management-System
A Python-based Hostel Room Management System using functions and dictionaries to manage hostel operations. This project allows users to assign rooms, vacate rooms, view all rooms, search by room or student, and list vacant rooms. It’s a simple, easy-to-understand application perfect for beginners learning Python fundamentals and practical coding.
A simple Python-based Hostel Room Management System using functions and dictionaries. This project helps manage hostel operations efficiently, including assigning rooms, vacating rooms, viewing room status, searching for students or rooms, and listing vacant rooms. It’s a beginner-friendly project that demonstrates practical Python coding skills and menu-driven program design.

**Features**
Assign rooms to students
Vacate rooms
View all rooms and their current status
Search by room number or student name
Show all vacant rooms
Simple and user-friendly menu interface

**Technologies Used**
Python (Functions & Dictionaries)
